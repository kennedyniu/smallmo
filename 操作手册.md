# 小沐记账小程序用户操作手册

## 📱 一、小程序简介

**小沐账本小程序**是一款专为个人设计的轻量级记账工具，无需登录，所有数据存储在您的手机本地。它提供完整的记账、分析和管理功能，帮助您轻松掌握个人收支状况。

### 主要特点
- 📊 **四个核心页面**：明细、图表、记账、我的
- 📁 **多账本管理**：支持创建多个账本（如“个人账本”、“家庭账本”）
- 🏷️ **分类自定义**：预置常用分类，支持添加/删除个性化分类
- 📈 **可视化图表**：饼图展示收支比例，柱状图显示月度趋势
- 🎨 **多主题风格**：5种精心设计的视觉主题
- 📥 **数据迁移**：支持CSV格式导入/导出，方便备份和跨设备同步

## 🚀 二、快速开始

### 1. 打开小程序
1. 在微信中搜索“小沐记账”或扫描开发者提供的二维码
2. 首次使用会请求存储权限，请点击“允许”以保存数据

### 2. 首次使用设置
- 进入后，系统会自动创建一个“默认账本”
- 预置了常用的收支分类（餐饮、交通、购物、工资等）
- 您可以直接开始记账，也可以先到“我的”页面进行个性化设置

## 🏠 三、主界面介绍

小程序底部有四个导航标签：

| 图标 | 名称 | 功能 |
|------|------|------|
| 📋 | **明细** | 查看所有账单列表，支持筛选和编辑 |
| 📊 | **图表** | 可视化分析收支数据 |
| ➕ | **记账** | 快速添加收入或支出记录 |
| 👤 | **我的** | 设置、账本管理、分类管理、数据导入导出 |

## 📋 四、明细页面

**功能**：查看历史账单，支持筛选，编辑，删除操作

### 1. 查看账单
- 页面按时间倒序显示所有账单（最新在最上面）
- 每条记录显示：分类图标、金额、日期、备注
- 支出显示为红色（-），收入显示为绿色（+）

### 2. 筛选功能
- **按账本筛选**：点击顶部账本名称，选择要查看的账本
- **按月份筛选**：点击月份选择器，查看特定月份的账单

### 3. 账单操作
- **编辑账单**：点击某条账单 → 修改信息 → 保存
- **删除账单**：左滑某条账单 → 点击“删除” → 确认

## 📊 五、图表页面

**功能**：通过可视化图表分析收支情况。

### 1. 收支统计（饼图）
- 显示当前所选账本和时间的收支比例
- **支出饼图**：各支出分类的金额占比
- **收入饼图**：各收入分类的金额占比

### 2. 月/年度趋势（柱状图）
- 显示年/月的收支趋势
- 蓝色柱子：收入金额
- 橙色柱子：支出金额

### 3. 图表筛选
- **选择账本**：可单选或多选账本进行对比分析
- **选择时间**：支持选择年或者月

## ➕ 六、记账页面

**功能**：快速记录收入或支出。

### 1. 记账流程
1. **选择类型**：点击顶部“支出”或“收入”切换类型
2. **选择分类**：从分类网格中选择一个分类（每行4个图标）
3. **输入金额**：输入金额（支持小数点），支持加减运算
4. **添加详情**：
   - **备注**：可选，记录详细信息
   - **日期**：点击日期选择器，默认为当天
   - **账本**：点击账本选择器，选择要记入的账本
5. **保存记录**：
   - **完成**：保存并返回明细页面
   - **下一笔**：保存当前记录，清空表单继续记下一笔

### 2. 分类管理（在记账页面）
- **添加分类**：点击分类网格最后的“+”按钮 → 输入名称 → 选择图标 → 保存
- **删除分类**：长按已有分类 → 点击“删除” → 确认（仅限自定义分类）
  ⚠️ **注意**：如果分类下有数据会提示迁移到其他分类
- **排序分类**：长按分类拖动调整顺序

## 👤 七、我的页面

**功能**：系统设置和数据管理。

### 1. 主题设置
- 点击“主题风格”进入选择页面
- 5种主题可选：
  - **简约现代**（默认）：蓝色系，清晰简洁
  - **暗色主题**：深色背景，护眼模式
  - **男性商务**：深蓝配色，沉稳专业
  - **女性优雅**：粉色系，温柔雅致
  - **可爱萌系**：橙色系，活泼可爱
- 选择后立即生效，无需重启小程序

### 2. 账本管理
- **查看账本**：显示所有已创建的账本
- **新建账本**：点击“+” → 输入账本名称 → 确定
- **切换账本**：点击账本名称切换默认账本
- **删除账本**：向左滑动账本 → 点击“删除” → 确认
  ⚠️ **注意**：如果账本下有数据会提示迁移到其他账本

### 3. 数据导入导出
#### 📤 导出数据
1. 点击“导出数据”
2. 选择时间范围：
   - **本月**：当前月份
   - **上月**：上一个月份
   - **本年**：当前年份
   - **上年**：上一个年份
   - **自定义**：手动选择开始和结束日期
3. 选择账本：可多选
4. 保存到手机

#### 📥 导入数据
1. 点击“导入数据”
2. 选择手机中的CSV文件
3. 系统自动解析并导入

### 5. 关于与反馈
- 查看小程序版本信息
- 提供反馈渠道

## 🔄 八、数据备份与迁移

### 建议的备份策略
1. **定期导出**：每月底导出当月数据，保存到电脑或云盘
2. **更换手机前**：导出全部数据，在新手机上导入
3. **清理数据前**：确保已备份重要数据


## ❓ 九、常见问题

### Q1：数据安全吗？会上传到服务器吗？
**A**：所有数据都存储在您的手机本地，不会上传到任何服务器。请定期导出备份以防误删。

### Q2：能同步到多台设备吗？
**A**：目前不支持自动同步。您可以在A设备上导出数据，然后在B设备上导入。

### Q3：分类可以自定义图标吗？
**A**：目前支持从预设图标库中选择，暂时不支持上传自定义图标。

### Q4：删除账本后能恢复吗？
**A**：不能。删除账本会永久删除该账本下的所有账单，请谨慎操作。

### Q5：支持多人共同记账吗？
**A**：这是个人记账工具，不支持多人实时协作。但可以为每个成员创建独立的账本。

### Q6：如何彻底清除所有数据？
**A**：在微信中找到小程序 → 点击右上角“…” → “设置” → “清空缓存”。⚠️ 此操作不可逆！
