小沐账本 (smallmo)

一款简洁实用、支持 AI 智能记账的微信小程序。告别繁琐的输入，一句话搞定记账！

✨ 功能特性

🤖 AI 智能记账

支持自然语言输入（例如：“中午吃肯德基花了85元”）。

自动识别金额、分类、备注，一键生成账单。

对接后端 AI 解析接口，准确率高。

📊 多维度数据统计

图表展示：支持按年、月查看收支趋势（柱状图）。

分类占比：支持饼图查看各类别消费占比。

数据筛选：可按支出/收入类型筛选，支持选择特定账本。

📁 多账本管理

支持创建多个独立账本（如：个人账本、家庭账本、旅行账本）。

支持账本切换、重命名及删除。

🎨 个性化主题

内置 5 种精美主题：简约现代、暗色主题、女性优雅、男性商务、可爱萌系。

主题实时切换，全局生效。

💾 数据安全

默认使用本地存储，无需注册，保护隐私。

支持开启云同步（可选），多设备无缝衔接。

🛠 技术栈

前端：微信小程序原生开发 (WXML, WXSS, JavaScript)

架构：组件化开发，全局状态管理

存储：

本地：wx.setStorageSync(Utils封装)

云端：微信云开发 (CloudBase)

AI 服务：Node.js + Express 后端接口

🚀 快速开始
1. 克隆项目
bash
git clone https://github.com/kennedyniu/smallmo.git
cd xiaomu-account
2. 导入微信开发者工具

打开 微信开发者工具。

选择 导入项目。

填入项目目录，AppID 可使用测试号或你自己的小程序 AppID。

3. 配置 AI 接口（必需）

打开 pages/ai/ai.js，确认后端接口地址是否正确（目前对接的是远程服务）：

javascript
// pages/ai/ai.js
wx.request({
  url: 'https://你的域名/ai/parse', // 当前使用的接口
  method: 'POST',
  // ...
});
4. 运行项目

点击 编译​ 按钮，即可在模拟器或真机上体验。

💡 使用指南
AI 记账

点击底部导航栏的 “AI”​ 图标。

在输入框输入自然语言，例如：

今天午饭吃了30块钱

打车去机场花了120元

买衣服消费500

点击 发送，系统会自动识别并存入账单。

切换主题

进入 “我的”​ 页面。

点击 “主题风格”。

选择喜欢的颜色，页面会立即切换。

查看报表

进入 “图表”​ 页面。

选择年份和月份。

点击上方的 支出/收入​ 切换数据。

点击柱状图的柱子或分类条形，可跳转到对应明细。

🔧 配置说明
本地存储 (storage.js)

所有账单数据默认存储在用户手机本地，数据结构如下：

javascript
{
  bills: [ // 账单数组
    {
      id: "unique_id",
      amount: 30,
      categoryId: "cat_123",
      type: "expense",
      date: "2026-06-12T10:00:00.000Z",
      remark: "午饭",
      accountId: "acc_456"
    }
  ],
  accounts: [ // 账本数组
    { id: "acc_456", name: "个人账本" }
  ],
  categories: { // 分类
    expense: [{ id: "cat_123", name: "餐饮" }],
    income: []
  }
}
