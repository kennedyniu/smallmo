# 小沐账本微信小程序 (sheepy-books)

一个功能完善的个人记账微信小程序，无需登录，数据本地存储。

## 功能特性

### 1. 四个主要页面
- **明细** - 账单列表展示，支持按账本、月份筛选
- **图表** - 收支统计饼图、月度趋势柱状图
- **记账** - 快速记账，支持支出/收入分类
- **我的** - 设置管理，包括主题、账本、分类、数据导入导出

### 2. 账本管理
- 支持创建多个账本
- 快速切换账本
- 删除账本（同时删除该账本下所有账单）

### 3. 分类管理
- 预置收支分类（餐饮、交通、购物、工资等）
- 支持自定义添加/删除分类
- 分类图标+名称

### 4. 账单管理
- 添加收入/支出记录
- 选择分类、日期、备注
- 编辑/删除账单

### 5. 数据导入导出
- **导出**：支持选择时间范围（本月/上月/本年/上年/自定义）和多个账本
- **导入**：支持CSV格式，根据id唯一识别，已存在的id不重复添加

### 6. 多主题风格
- 简约现代（默认）
- 暗色主题
- 男性商务款
- 女性优雅款
- 可爱萌系款

用户可在"我的"页面设置默认主题风格。

## 技术栈

- 原生微信小程序
- 本地存储（wx.getStorageSync / wx.setStorageSync）
- Canvas绘制图表

## 使用说明

1. 下载微信开发者工具
2. 导入项目目录
3. 在开发者工具中预览或真机调试

## 数据结构

```javascript
// 本地存储结构
{
  "accounts": [{id, name, createTime}],  // 账本列表
  "currentAccount": "accountId",          // 当前账本ID
  "categories": {                          // 分类列表
    expense: [{id, name, icon}],
    income: [{id, name, icon}]
  },
  "bills": [{                              // 账单列表
    id, accountId, date, amount,
    type: 'expense'|'income',
    categoryId, remark, createTime
  }],
  "settings": {theme: 'simple'}           // 设置
}
```
